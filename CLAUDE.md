# 一般

- コミュニケーションや記述は**日本語**で行い、思考は**英語**で行ってください。
- 指示がない限り、Python などのコード実行を禁止します。
- ファイルの読み書きは UTF‑8(no BOM)で行ってください。
- エージェントは指示を与えられた際、必ず方針と計画を/docs/log/YYYYMMDD*HHmmSS*{overview}.md として作成して、実行してよいか指示を待つ。
  - タイムスタンプは必ず以下のコマンドで取得すること（JST）: `TZ='Asia/Tokyo' date '+%Y%m%d_%H%M%S'`
  - 実行中に提示した方針や計画と異なる動作を行う必要が出た場合は、再度方針と計画を作成して指示を待つ。

# サブエージェント

- git-agent: git 操作を行う際には必ずこのエージェントを利用してください。
  - git-agent の報告後、必ず`git status`または`git log`で結果を検証すること
  - エージェントの報告を鵜呑みにせず、実際の操作結果を確認すること
- lead-researcher: 包括的な調査や複数ソースからの情報収集が必要な場合に利用してください。
- research-assistant: 調査が必要な場合にはこのエージェントを利用して最新情報を収集してください。
